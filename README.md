# AutoDCS
충남대학교 사이버캠퍼스 과제, 자료실, 콘텐츠 등등 TodoList를 알려주는 카카오 챗봇입니다.
*OldVersion(2023)안에 있는 파일들은 이 프로젝트의 초창기 버전으로 현재와는 완전히 다른 내용입니다.*
